# Credit Risk Classification Model

This project builds a machine learning model to predict whether a customer will default on a loan using the UCI Credit Card dataset.

## Models Used
- Logistic Regression
- Random Forest

## Project Features
- Data preprocessing and cleaning
- Feature engineering
- Model training and evaluation
- Confusion matrix visualization
- ROC curve analysis

## Dataset
UCI Credit Card Default Dataset (Kaggle)
